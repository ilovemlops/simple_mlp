# Simple MLP as of micrograd
This repository consists of a single Jupyter Notebook to create MLP and visualize it using `graphiz`

## How to run
1. Create virtual environment
```
python -m venv .venv
```

2. Activate virtual environment
```
source .venv/bin/activate
```

3. Install dependencies
```
pip install -r requirements.txt
```

4. Install `graphiz` (Mac Example)
```
brew install graphviz
```

4. Run the code in the notebook
[simple_mlp.ipynb](simple_mlp.ipynb)
